# 中文版：美泰 IQC 进料检验与追踪系统 标准操作手册

本系统采用“双角色物理隔离”与“离线文件接力”的工作模式。**检验员**与**质量主管**在各自的电脑上独立操作，通过导出/导入 JSON 文件实现单据的流转与数据合并。

---

## 👨‍🔧 第一部分：IQC 检验员操作指南 (制单、化学补录与物理初判)

### 步骤 1：来料录入与【化学测试频次】智能判定

1. 打开系统网页，在【1. 新来料批次输入与智能拦截判定】区域填写交货单信息。
2. **快捷录入**：在输入“供应商代码”或“供应商名称”时，系统会自动模糊匹配，并带出该供应商的【美泰供应商等级】。
3. 依次选择“物料大类”和“物料细类”。
4. 填写完毕后，系统底部会自动弹出关于**化学测试的红绿灯判定结果**：
* 🚨 **红标 (必须测试)**：代表该物料到了化学测试周期（如 12个月/6个月已过期），或属于非指定供应商的每批必检物料。
* 🔒 **绿标 (无需测试)**：代表化学测试仍在有效期内，系统已自动锁死输入框，并智能带出上一份合格化学报告的日期与编号。


5. 确认无误后，点击 **[💾 保存批次入库]**。单据会自动进入下方的管理台账。

### 步骤 2：【化学测试】报告信息快捷补录

在实际操作中，红标“必须测试”的物料往往在入库时还没有拿到化学化验报告。系统支持在台账中快捷补录：

1. 当化验室出具了化学报告后，在台账列表中找到对应批次。
2. **双击**该行的 **[化学报告日期]** 单元格，会变成一个日历选择器，选好日期后按 `Enter` 键保存。
3. **双击**该行的 **[化验报告编号]** 单元格，输入化验单号（如 C-2026-XXXX）后按 `Enter` 键保存。
4. *智能同步提醒：如果这个物料后续有跟单到货，只要您补录了主批次的化学报告，后续跟单的记录会自动同步更新并锁死，无需重复录入。*

### 步骤 3：【物理检验】实测数据录入与初判

1. 在台账中点击对应单据最右侧的 **[⏳ 待检验]** 按钮，打开物理检验窗口。
2. 系统已根据您选择的物料细类，**自动匹配好了对应的物理检验项目和标准规格**。
3. 在表格中逐行录入物理实测的“抽样数”、“次品数”和“实测数据”。
* *防呆提示：上方输入 Ac (接收数) 后，系统会自动计算出 Re (拒收数)。*


4. 滚动到表格下方，在 **[📊 检验员提交初判]** 处选择物理检验结论（合格/不合格）。
5. 在最下方的 **[📝 检验员备注]** 框内补充现场异常描述。
6. 点击底部的 **[💾 仅保存数据]**。此时台账状态会变为灰蓝色的 `[⏳ 待审批]`。

### 步骤 4：打包送审 (发给主管)

1. 在台账列表左侧，**勾选**需要送审的单据（可一次勾选多个）。
2. 点击台账上方的 **[📤 提取导出 [选中行 JSON] 用于协同]** 按钮。
3. 系统会自动下载一个文件，命名格式为 `待批准-年月日-时分秒.json`。
4. 将此文件通过微信、钉钉或邮件发送给质量主管。

### 步骤 5：接收批复与打印报告

1. 收到主管发回的批复文件后，点击页面右上角的 **[📥 智能导入/合并 JSON]**，选择该文件导入。
2. 系统会自动将审批结果无损合并到您的台账中，相关单据状态会变绿显示 `[✅ 已批准]`。
3. 双击台账打开该单据的物理检验窗，滚动到底部，点击 **[🖨️ 打印最终审批报告 (In PDF)]** 按钮直接打印。
* *打印提示：系统已对排版进行锁定，检验员备注框会自动固定为标准的 8 行高度，不会出现排版错乱。保存的 PDF 文件会自动以该物理报告的编号命名。*

## 👨‍💼 第二部分：质量主管操作指南 (审批与回传)

主管审批需要在专属的“主管模式”下进行，以解锁最高修改与打印权限。

### 步骤 1：解锁隐藏主管模式

1. 打开系统网页后，将鼠标移至页面最上方中间的大标题 **“美泰 IQC 进料检验与追踪系统”**。
2. **快速双击**该大标题，系统会弹出鉴权密码框。
3. 输入主管专属密码：`ABC123456`，点击确定。
4. 鉴权成功后，标题旁会出现红色的 `[主管模式激活]` 徽章，代表您的审批与打印权限已完全解封。

### 步骤 2：导入并审核检验单

1. 收到检验员发来的 `待批准-xxx.json` 文件后，点击页面右上角的 **[📥 智能导入/合并 JSON]**，载入该文件。
2. 在下方台账中找到状态为 `[⏳ 待审批]` 的单据，点击它打开物理检验窗。
3. 核对上方的“化学测试”报告信息是否齐全，查阅物理实测数据、初判结果及备注信息。

### 步骤 3：填写审批意见并签批

1. 滚动到检验窗最下方的绿色区域 **【✍️ IQC 主管最终审批区】**。
2. 在 **[审批备注]** 框内填写您的审核意见、特采原因或退货说明（此框位于最上方，方便填写长段落）。
3. 在下方选择 **[审批结论]**（同意接收 / 特采接收 / 拒收退货）。
4. 确认 **[主管电子签名]**（QC主管/QA经理）和 **[审批日期]**。
5. 点击底部的 **[💾 仅保存数据]**，完成审批。

### 步骤 4：回传已批复数据

1. 在台账中，勾选您刚刚批复完成的单据（状态已变为绿色的 `[✅ 已批准]` 等）。
2. 点击台账上方的 **[📤 提取导出 [选中行 JSON] 用于协同]** 按钮。
3. 系统会自动下载名为 `已批准-年月日-时分秒.json` 的文件。
4. 将此文件通过微信或邮件发还给检验员，由检验员导入其电脑以完成闭环。

### 步骤 5：全量防灾备份 (每周执行)

为防止浏览器缓存清理导致数据丢失，主管需定期进行系统全量备份：

1. 点击页面右上角的 **[📤 导出 JSON 备份]** 按钮。
2. 系统将下载包含所有历史台账、所有单据和供应商名录的全局文件，命名为 `Mattel_Integrated_DB_Backup_年月日_时分秒.json`。
3. 请将此文件保存在公司公共网盘或加密 U 盘中。若遇电脑更换，在空系统中导入此文件即可一秒恢复全部数据。
